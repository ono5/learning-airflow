# learning-airflow

## Install
[Astro CLI](https://docs.astronomer.io/astro/cli/install-cli)

```
brew install astro
```

## Setup

```
astro dev init
```

## Install

```
pip install "apache-airflow[celery]==2.5.0" --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-2.5.0/constraints-3.7.txt"
```

## Start

```
astro dev start
```

## Stop

```
astro dev stop
```
